# Abhay Tiwari Portfolio - Frontend

A modern, responsive portfolio built to showcase my projects and provide an interactive AI chat experience.

## Tech Stack
* **Framework:** Next.js (App Router)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion

## Key Features
* **AI Chat Interface:** Floating chat bubble allowing recruiters to "talk to my resume".
* **Responsive Design:** Optimized for all screen sizes using Tailwind CSS.
* **Production Projects:** Showcases my work at Label Lift, Therapist Maya, and NCC Attendance Tracker.

## Setup
1. Clone the repository.
2. Install dependencies: `npm install`
3. Update the API endpoint in the chat component to point to the live Render URL.
4. Run the development server: `npm run dev`
